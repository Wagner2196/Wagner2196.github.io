Principais Uso

Comando Descrição
git init Inicializa um novo repositório Git.
git add Adiciona arquivos ao staging area.
git commit Salva as mudanças no histórico do repositório.
git status Exibe o estado atual do repositório.
git log Mostra o histórico de commits.

Esses comandos são fundamentais para gerenciar um repositório Git, permitindo a criação de novos projetos,
adição e confirmação de mudanças,
monitoramento do estado atual do projeto e
revisão do histórico de alterações realizadas.
