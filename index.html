Comandos Git e Uso no Projeto

Configuração e Acesso ao Repositório GitHub

Para configurar o Git com seu usuário GitHub, siga os passos abaixo:

git config --global user.name "Seu Nome"
git config --global user.email "seuemail@example.com"
git config --global init.defaultBranch main

Para clonar um repositório GitHub:

git clone https://github.com/usuario/repositorio.git

Comandos Git

git init

Inicializa um novo repositório Git.

git init

git clone

Clona um repositório existente.

git clone https://github.com/usuario/repositorio.git

git branch

Gerencia branches no repositório.

git branch

Para criar uma nova branch:

git branch nome-da-branch

git checkout

Troca de branch ou restaura arquivos no repositório.

git checkout nome-da-branch

git status

Mostra o estado dos arquivos no repositório.

git status

git diff

Mostra as diferenças entre os arquivos no repositório.

git diff

git add

Adiciona arquivos ao índice (staging area).

git add arquivo.txt

Para adicionar todos os arquivos modificados:

git add .

git commit

Registra as mudanças no repositório.

git commit -m "Mensagem do commit"

git push

Envia as mudanças para o repositório remoto.

git push origin nome-da-branch

git pull

Atualiza o repositório local com as mudanças do repositório remoto.

git pull origin nome-da-branch

git revert

Desfaz um commit específico.

git revert id-do-commit

git merge

Combina mudanças de diferentes branches.

git merge nome-da-branch

git stash

Armazena mudanças temporariamente.

git stash

Para aplicar mudanças armazenadas:

git stash apply

git rm

Remove arquivos do repositório.

git rm arquivo.txt

git config

Configura opções de usuário e repositório.

git config --global user.name "Seu Nome"

git reset

Reseta mudanças no índice e na árvore de trabalho.

git reset --hard

git remote

Gerencia repositórios remotos.

git remote add origin https://github.com/usuario/repositorio.git

git fetch

Baixa mudanças do repositório remoto sem aplicá-las.

git fetch origin

git show

Mostra detalhes sobre um objeto Git (commit, tag, etc.).

git show id-do-commit

git help

Mostra ajuda sobre comandos Git.

git help comando

git rebase

Reaplica commits em cima de outra base.

git rebase nome-da-branch

git blame

Mostra o autor de cada linha de um arquivo.

git blame arquivo.txt

git tag

Marca commits com uma tag específica.

git tag -a v1.0 -m "Versão 1.0"
